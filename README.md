# Jmix TinyMCE Editor 

This add-on provides support for [TinyMCE WYSIWYG Editor](https://www.tiny.cloud/) through the [TinyMCE wrapper for Vaadin Flow](https://vaadin.com/directory/component/tinymce-for-flow) addon.
